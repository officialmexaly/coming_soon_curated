# Dreamscape

Next.js 16.2 App Router version of the Dreamscape coming soon landing page.

## Scripts

- Run `npm install` first
- `npm run dev` starts the Next.js dev server
- `npm run build` builds the app for production
- `npm run start` runs the production server
- `npm run lint` runs ESLint

## Countdown

The countdown start time is stored server-side in `server-data/countdown.json` and shared by all users served from the same Node process or instance.
